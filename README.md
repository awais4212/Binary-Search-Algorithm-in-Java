## 📘 Overview
This Java program demonstrates the **Binary Search Algorithm** on a **sorted array**.  
It efficiently searches for a specific key by repeatedly dividing the array into halves,  
comparing the middle element, and adjusting the search range until the key is found.  

If the key exists, its **index** is displayed; otherwise, a message indicates that the  
element is **not present** in the array.

---

## ⚙️ Features
- 🧠 Implements an efficient search algorithm  
- 📈 Works seamlessly on sorted arrays  
- 💡 Simple and beginner-friendly logic  
- 🖥️ Command-line interface output  

---

## 🧩 Algorithm Steps
1. Sort the array (if not already sorted).  
2. Find the middle element.  
3. Compare the key with the middle element.  
4. If the key matches, return its index.  
5. If smaller, search the left half; if larger, search the right half.  
6. Continue until the element is found or the array is fully searched.  

---

## 🧠 Time Complexity
| Case | Complexity |
|------|-------------|
| **Best** | O(1) |
| **Average** | O(log n) |
| **Worst** | O(log n) |
